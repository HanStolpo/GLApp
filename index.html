Intro

Hakyll generated static site to for blog posts and slides for our computer graphics programming users group.

Examples

  • General pages go in the pages folder e.g. /pages/Example_Page.html
  • Posts go in the posts folder by date e.g. /posts/2014-08-27-Example_Post.html
  • Slides go in the slides folder by date e.g. /slides/2014-08-27-Example_Slides.html
  • Media files go in the media folder (just get copied to the site)
  • css files go in the css folder (just get copied to the site)
